# wabot-aq
Simple WhatsApp Bot

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> git clone https://github.com/Nurutomo/wabot-aq
> cd wabot-aq
> npm install
```
###### Run
```bash
> node . [<session name>] (session name is optional)
```
